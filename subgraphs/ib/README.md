# Index Betting Subgraph

### Set appropriate network:
Set the path inside the [config](config/config.js) to one of the networks defined inside the [config](config) folder.
```shell
let defaultPath = './mainnet/';
```

### Building the project

```bash
# run pre codegen
npm run precodegen
# run codegen
npm run codegen
# run build
npm run build
```
